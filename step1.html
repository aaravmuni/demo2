Form A SECTION 1 / IDENTITY BLOCK

Complete all required entries. Do not abbreviate unless instructed. Entries are case-significant.

Section 1 — Identity Block

* Primary Identifier Format: surname, comma, given name. Must match registry.
Secondary Identifier If none, leave blank. If blank, may be required later.
* Designation Code 4–8 characters. Normalised on entry.
Reference Number For internal cross-reference (Ref. 4.2).

Section 2 — Contact Block

* Contact Sequence Numeric contact sequence.
* Origination Date Enter as DD/MM/YYYY.
* Locale Code Refer to locale schedule. L09 is deprecated but selectable.
Classification       Default U. Change only if applicable.

Form A. Identity fields are protected and may be cleared on validation failure.
